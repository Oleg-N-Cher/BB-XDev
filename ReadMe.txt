XDev — многоцелевая среда разработки на Обероне-2 для
MS-DOS, Windows 32/64bit, Linux, ZX Spectrum и MSX
=====================================================

XDevLite или XDevFull — это хост-система разработчика,
необходимая для использования любой из подсистем.

Подсистема (или несколько) распаковывается из архива
в папку с установленным (или распакованным) XDev
(Lite или Full). Всё готово к работе.

После запуска XDev[Lite].exe вы увидите окно среды разработки.

Быстрый старт:

File -> Open -> Ваша подсистема -> Mod - примеры

F11 - скомпилировать; F12 - собрать и запустить.

Выделить мышкой название библиотечного модуля и:

XDev -> Show Definition - показать интерфейс данного модуля

Ctrl+G - навигация по коду (в Lite не присутствует)

Этого объёма знаний будет вполне достаточно для изучения
примеров и написания своих.


В качестве документации используйте публикации по языку Оберон,
а также будем рады ответить на ваши вопросы на форуме:

http://zx.oberon2.ru/forum

Среда содержит недоработки и известные проблемы. Любая ваша помощь,
участие или содействие очень важны для дальнейшего развития XDev.


Команда разработки XDev.
(с) Оберон-клуб «ВЄДАsoft», 2015


Целевые подсистемы/платформы
============================

ZXDev - подсистема для разработки программ для ZX Spectrum


==========================================================
XDev is a multi-target development environment in Oberon-2
for MS-DOS, Windows 32/64bit, Linux, ZX Spectrum and MSX
==========================================================

XDevLite or XDevFull is a developer's host-system what
you need to use any of the target subsystems.

A subsystem (or more) need to be unpacked from the archive
into the folder containing the installed (unpacked) XDev
(Lite or Full). Everything is ready to go.

After starting XDev[Lite].exe you'll see the development environment.

Quick start:

File -> Open -> Your Subsystem -> Mod - examples

F11 - compile; F12 - build and run.

Choose with mouse a name of a library module and:

XDev -> Show Definition - show interface of the module

Ctrl + G - code navigation (does not present in Lite)

Knowledge of this volume will be sufficient for the studying
examples and writing own.


As documentation, use the publications on Oberon language.
Also welcome to ask your questions at the forum:

http://zx.oberon2.ru/forum

( If you don't know Russian, you can use Google translator )


XDev environment contains defects and known problems. Any your help,
participation or assistance is very important for further development.


XDev development team.
